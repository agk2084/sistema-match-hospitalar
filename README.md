# Sistema de Similaridade para Encaminhamento de Pacientes

Projeto desenvolvido para a disciplina Projeto Interdisciplinar: Desenvolvimento de Software da Universidade Tuiuti do Paraná.

## Objetivo

Desenvolver um sistema capaz de recomendar o profissional de saúde mais adequado para atendimento em ambiente de pronto atendimento hospitalar utilizando técnicas de similaridade.

## Tecnologias
- Java
- Programação Orientada a Objetos
- Astah* UML

## Funcionalidades
- Leitura de profissionais via CSV
- Inserção de dados do paciente
- Cálculo de similaridade
- Geração de ranking
- Recomendação de profissional
